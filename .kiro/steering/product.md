# Product Overview

vibehuntr-agent is an AI-powered document retrieval and question-answering agent built with Google's Agent Development Kit (ADK). The agent provides intelligent responses by retrieving relevant information from indexed documents using Vertex AI Search and Vector Search capabilities.

## Core Capabilities

- **Document Retrieval**: Searches and retrieves relevant documents based on user queries using Vertex AI Search
- **RAG (Retrieval Augmented Generation)**: Combines retrieved context with LLM reasoning for accurate, grounded responses
- **Event Planning Domain**: Includes specialized models and services for event planning use cases (groups, users, events, feedback)
- **Production-Ready**: Includes monitoring, tracing, feedback collection, and deployment infrastructure

## Key Features

- Automated data ingestion pipeline for indexing documents
- Re-ranking with Vertex AI Rank for improved relevance
- Cloud Logging and OpenTelemetry tracing for observability
- Feedback collection and logging to BigQuery
- Streamlit playground for local testing
- CI/CD pipelines for automated deployment
